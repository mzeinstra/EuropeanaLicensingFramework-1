EDM Rights Selection Tool Recommendation

Based on your answers in the EDM Selection Tool it is recommended that you apply one of the following two Creative Commons licences to the digital object:

In both cases the object may not be modified by the user and the user will need to provide attribution.

Apply a Creative Commons Attribution NoDerivatives licence to allow third parties to use the object for commercial and non-commercial purposes you should by inserting the following rights value:

http://creativecommons.org/licenses/by-nd/3.0/

Apply a Creative Commons Attribution NonCommercial NoDerivatives licence to allow third parties to use the object for non-commercial purposes only you should by inserting the following rights value:

http://creativecommons.org/licenses/by-nc-nd/3.0/

Note: make sure that the selected licence terms are clearly communicated on your own website. You can find instructions on how to do this at Creative Commons.
